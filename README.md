Program ganjil genap berfungsi untuk menentukan apakah bilangan yang dimasukkan pengguna ganjil atau genap.
Cara kerjanya: program meminta input bilangan, lalu memeriksa sisa hasil bagi bilangan tersebut dengan 2 (bilangan % 2).
Jika sisa = 0 → genap, jika tidak → ganjil.
Hasilnya ditampilkan ke layar.
